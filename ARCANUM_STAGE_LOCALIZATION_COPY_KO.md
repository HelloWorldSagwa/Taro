# ARCANUM STAGE 한국어 카피 가이드

## Sprint 1 톤

ARCANUM STAGE의 한국어 문체는 `게임 UI의 명료함`과 `타로 상담의 다정한 확신`을 함께 사용한다. 사용자는 운세 앱의 고객이 아니라, 루나와 함께 오늘의 무대에 앉은 플레이어로 대한다.

## 문체 원칙

- 짧은 버튼 문구: 행동이 바로 보이게 쓴다. 예: `오늘의 카드 보기`, `테이블로 돌아가기`
- 상담 대사: 단정하지 않고 부드럽게 안내한다. 예: `살펴보세요`, `정리됩니다`, `괜찮아요`
- 불안 조장 금지: 운명 확정, 관계 강요, 공포 유발 표현을 피한다.
- 카드 해석: 전문 용어보다 오늘의 행동으로 옮길 수 있는 문장으로 마무리한다.
- 게임 무대감: `테이블`, `아르카나`, `열리다`, `응답하다`처럼 조작과 연출이 느껴지는 단어를 사용한다.

## Sprint 1 네이밍

| 흐름 | 한국어 명칭 | 사용 의도 |
|---|---|---|
| Main screen | 아르카눔 스테이지 | 브랜드명은 한국어 표기로 첫 화면에 크게 노출 |
| Profile setup | 첫 상담 준비 | 아직 구현 전이지만, 프로필 생성의 부담을 낮추는 이름 |
| Today's reading CTA | 오늘의 카드 보기 | 무료 1장 리딩의 목적을 즉시 전달 |
| Card selection prompt | 눈길이 머무는 카드 | 사용자의 직감 선택을 부드럽게 유도 |
| Revealed card result | 오늘 열린 아르카나 | 카드 공개의 의식감과 결과성을 함께 전달 |
| Tarot master | 타로 마스터 루나 | 캐릭터 이름을 상담 주체로 고정 |
| Result note | 루나의 상담 노트 | 결과 텍스트를 판정문이 아닌 상담 기록처럼 전달 |

## Sprint 1 대사 샘플

- 홈: `어서 와요. 마음이 준비되면, 오늘 당신에게 필요한 첫 카드를 함께 열어볼게요.`
- 선택: `서두르지 않아도 괜찮아요. 시선이 자꾸 돌아가는 곳에 잠시 손을 두세요.`
- 결과: 카드별 `MasterLine`은 루나가 카드의 첫 인상을 말하는 한 문장으로 유지한다.
